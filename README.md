# Customer Behavior & Trends Analytics

An end-to-end data analytics project replicating a real-world company analytics workflow — from raw data ingestion to an interactive Power BI dashboard.

---

## Project Overview

This project analyses customer purchasing behaviour across demographics, product categories, and sales channels to deliver actionable marketing insights. The full pipeline covers Python EDA, SQL segmentation, cohort analysis, and an interactive Power BI dashboard.

**Key finding:** Young adults (18–24) are the highest revenue-generating demographic — validating campaign prioritisation for this segment.

## How to Run

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Run EDA & Feature Engineering

```bash
cd python
python 01_eda_feature_engineering.py
```

Outputs:
- `data/customer_behavior_enriched.csv`
- `docs/eda_overview.png`

### 3. Run Cohort & Revenue Analysis

```bash
python 02_cohort_revenue_analysis.py
```

Outputs:
- `docs/cohort_analysis.png`
- `docs/monthly_cohort_trend.png`
- `docs/discount_impact_age.png`

### 4. Run SQL Queries

```bash
cd ../sql
python run_queries.py
```

Outputs:
- 6 CSV files in `data/sql_outputs/` — ready for Power BI import

### 5. Build Power BI Dashboard

Follow the guide in `powerbi/POWERBI_SETUP.md` to:
- Import all CSVs
- Create DAX measures
- Build 5 dashboard pages with slicers and KPI cards

## Key SQL Techniques Used

- `WITH` (CTEs) for readable, layered query logic
- `RANK() OVER (PARTITION BY ...)` — category-level revenue ranking
- `NTILE(5) OVER (...)` — RFM quintile scoring
- `SUM(...) OVER (ORDER BY ...)` — running cumulative revenue
- `LAG(...) OVER (...)` — month-over-month growth calculation
- `CASE WHEN` cohort classification (New / Returning / Loyal)
- `JULIANDAY()` for date arithmetic (recency in days)

---

## Key Python Techniques Used

- `pd.cut()` — age group binning into named segments
- `LabelEncoder` — encoding categoricals for downstream analysis
- `pivot_table()` — category × age group revenue matrix
- `groupby().agg()` — multi-metric cohort summaries
- `merge()` — joining order counts back to transaction level
- `seaborn heatmap / scatter / barplot` — EDA visualisations

## Business Insights

1. **Young Adults (18–24) drive the highest total revenue** — prioritise this cohort for campaign spend
2. **Loyal customers (5+ orders) generate ~40% of revenue** from only ~15% of the customer base
3. **Electronics & Home & Kitchen** are the top two revenue categories
4. **Online channel accounts for ~50% of all transactions** — digital-first campaigns recommended
5. **Discount utilisation averages 18–22%** — young adults respond more to discount-driven campaigns
6. **RFM segmentation** reveals a significant "At Risk" segment that can be re-engaged with targeted offers
